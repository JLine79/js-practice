9. Write a JavaScript program to calculate days left until next Christmas.

  • Ok that was harder than I thought it would be!
  • Remember if you want to figure out the number of days in a year you have to start from knowing the number of milliseconds since 1970
  • If you want to work out the number of days using milliseconds you have to convert milliseconds into one day like this
  • 1000 milliseconds (per second) * 60 seconds (per minute) * 60 minutes (per hour) * 24 hours (per day)
